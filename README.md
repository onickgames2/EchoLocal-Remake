# 🎵 EchoLocal - Player de Música Local

O **EchoLocal** é uma plataforma de streaming de música focada em arquivos locais, projetada para oferecer uma interface fluida, organizada e com animações suaves.

## ✨ Diferenciais
- **Interface Soft:** Design focado em animações "macias" e transições de fade.
- **Organização Inteligente:** Categorias para álbuns, artistas e playlists.
- **Player Customizado:** Controle total sobre a reprodução com sistema de progresso visual.
- **Estética Moderna:** UI limpa com efeitos visuais dinâmicos.

---

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 (Flexbox/Grid) e JavaScript puro.
- **Animações:** Keyframes para efeitos de entrada e partículas.
- **Áudio:** Web Audio API para controle de reprodução.
